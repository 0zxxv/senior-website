## ğŸ“– About the Project

This is the official marketing and informational website for **Wessal Education (Wessal Edu)**, an integrated intelligent system designed for future schools in the Kingdom of Bahrain. The website serves as a comprehensive platform to communicate the system's value proposition, features, and benefits to various stakeholders.

### Key Objectives

- Present the Wessal Edu system professionally to potential adopters
- Clearly communicate the 5 core pillars of the system
- Target three key stakeholder groups (Parents, Teachers, Administrators/MoE)
- Showcase the technology stack and system components
- Provide an easy way for interested parties to request demos

## ğŸš€ Getting Started

### Prerequisites

- A modern web browser (Chrome, Firefox, Safari, Edge)
- A local web server (optional, for development)

### Installation

1. **Clone or download the project**
   ```bash
   git clone https://github.com/your-username/wessal-edu-website.git
   cd wessal-edu-website
   ```

2. **Open in browser**
   - Simply double-click `index.html`, or
   - Use a local server:
     ```bash
     # Using Python
     python -m http.server 8000
     
     # Using Node.js (with npx)
     npx serve
     
     # Using VS Code Live Server extension
     # Right-click index.html â†’ Open with Live Server
     ```

3. **View the website**
   - Navigate to `http://localhost:8000` (or your server's port)
